# Contributing

We're always looking for feedback! We follow the below feature inclusion process to evaluate new ideas for inclusion into the Design System.
This helps us ensure we're delivering the highest value features to the community.

<div>
  <img src="/images/feature-inclusion-process.png" />
</div>

There are a number of ways you can contribute feedback such as requesting features, reporting bugs, and participating in our discussions space.
Find an issue or have an idea? [Submit an issue](https://github.com/Banno/jack-henry-design-system/issues).

Join the conversation in our [GitHub discussion board](https://github.com/Banno/jack-henry-design-system/discussions) or via our internal slack #org-design-system channel.
