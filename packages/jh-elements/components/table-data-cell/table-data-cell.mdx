# Table Data Cell

[Guidelines](https://jackhenry.design/pages/components/table-data-cell/)

<hr />

The table data cell is used to display one cell of table data. Table data cells can contain text or other content and have to be placed inside `<jh-table-row>`s.

## Usage

Import the `<jh-table-data-cell>` component:

```javascript
import '@jack-henry/jh-ui/components/table-data-cell/table-data-cell.js';
```

To use `<jh-table-data-cell>`, place it in the default slot of the relevant `<jh-table-row>`: 

```html
<jh-table> 
    // [...]
  <jh-table-row>
    <jh-table-data-cell>Cell 1</jh-table-data-cell>
    <jh-table-data-cell>Cell 2</jh-table-data-cell>
    <jh-table-data-cell>Cell 3</jh-table-data-cell>
  </jh-table-row>
  // [...]
</jh-table>
```

## Accessibility
